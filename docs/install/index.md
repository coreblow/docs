# Installation

## npm Install

```bash
npm install -g coreblow
coreblow onboard
coreblow gateway run
```

## Quick Install Script

```bash
curl -fsSL https://coreblow.com/install.sh | bash
```

## From Source

```bash
git clone https://github.com/coreblow/coreblow.git
cd coreblow
pnpm install
pnpm coreblow --help
pnpm coreblow gateway run
```
